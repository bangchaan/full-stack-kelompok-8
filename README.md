# full-stack-kelompok-8
Tema & Konsep Desain
Website ini mengangkat tema kerajinan tangan Nusantara dengan estetika earth tone yang hangat dan elegan, memadukan warna terracotta, cokelat tua, krem, dan aksen emas. Font yang digunakan adalah Playfair Display (serif klasik untuk judul) dipadu DM Sans (modern dan bersih untuk teks), menciptakan nuansa premium namun tetap membumi dan autentik.
